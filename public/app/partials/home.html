Project Path

An experiment in visualizing career paths

What is Project Path?

Career paths are no longer linear and predictable. These days, people often jump from one industry to another. How do they do that? Our guess is that there are discernable patterns, and we want to find those out.

Project Path attempts to abstract career paths from the powerful data available on LinkedIn. All the data is there, but it's not easy to search for paths, as opposed to people.

We built a data visualization tool to help you view your LinkedIn network in a new way. We've only just begin this project and hope to bring more complexity and insights in the near future.
